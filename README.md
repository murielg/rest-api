## Rails 5 API + JSON API 

Active Model Serializer and rack-cors

### What you'll need

- Rails 5 and ActiveModel Serializer. 
- Ruby version >= **2.3.4**
- Your own server/database

### You Know What To Do

- `$ bundle install`

```
$ rails db:create
$ rails db:migrate
$ rails db:seed
```

- `$ rails s`

- Navigate to `http://localhost:3000`
